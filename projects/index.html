Projects

I build automation and integration tooling around telecom deliverables, HR systems, finance workflows, and Quickbase apps — mostly using Python, Google Cloud Functions, Quickbase, and Excel/Box integrations.

This page groups my work into themes and shows how individual cloud functions, pipelines, and templates fit into bigger systems.

HR & Timekeeping Automations

Automations that keep HR, availability, and timekeeping data accurate without manual entry.

  • Kronos Weekly Timesheets Sync – Cloud Function that updates weekly HR timesheets in Quickbase using Kronos exports.
  • Kronos Holiday Calendar Automation – Ensures holiday rules and pay-period logic stay consistent across HR systems.
  • Employee Out-of-Office → HR Sync – Weekly automation that reads OOO/off-work emails and updates HR records.
  • Kronos Connector Pipeline – Integration pipeline linking Kronos, Quickbase, and Cloud Functions for seamless data flow.

  • verizon_ariba_core_sync – Keeps Quickbase “core” data aligned with Ariba updates.
  • PO_Line_Codes, POLineItems, pricing_table, Quotes_and_POR – Automated bulk upload jobs.
